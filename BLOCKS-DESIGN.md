# Варианты дизайна блоков для "Культура заводской сборки" и т.д

## Проблема с текущим дизайном (Unlike Any Tool)

**Сейчас:** Bento-сетка с клавиатурой, молнией, логотипом, меню  
**Почему не подходит:**

- Клавиатура/меню — абстракция для софта, не для авто
- Карточки статичные (только hover-эффект на кнопке)
- Нет "эволюционной траектории" (Identity-Based)

---

## Вариант 1: Expandable Cards (как в Apple)

**Концепция:** 3 карточки в ряд. При клике — разворачивается на весь экран с деталями.

### Визуал

```
[Культура заводской сборки] [Цифровой суверенитет] [Рациональная защита]
     ↓ клик
[════════════════════════════════════════════════════════════════]
│ Культура заводской сборки                                      │
│ Для тех, кто ценит инженерную точность.                        │
│                                                                 │
│ • Вы понимаете ценность каждого заводского шва                 │
│ • Стандарт инсталляции без разбора дверей                      │
│ • Сохранение целостности узлов и дилерской гарантии            │
│                                                                 │
│ [Визуал: 3D-модель двери с подсветкой узлов]                   │
[════════════════════════════════════════════════════════════════]
```

### Технологии

- Framer Motion (`layout` prop для плавного разворачивания)
- Spline/Three.js для 3D-визуалов (опционально)
- Glassmorphism для карточек

### Плюсы

- ✅ Интерактивность (клик → детали)
- ✅ Премиальный эффект (как в Apple)
- ✅ Фокус на одной карточке (не перегружает)

### Минусы

- ⚠️ Требует 3D-визуалов (или можно заменить на иконки/иллюстрации)
- ⚠️ На мобилке — вертикальный стек (но это ок)

---

## Вариант 2: Scroll-Triggered Reveal (как в Linear "Features")

**Концепция:** 3 карточки появляются по очереди при скролле. Каждая — с анимацией и визуалом.

### Визуал

```
[Скроллишь вниз]
     ↓
[Культура заводской сборки] ← появляется с fade-in + slide-up
     ↓
[Цифровой суверенитет] ← появляется через 0.2s
     ↓
[Рациональная защита] ← появляется через 0.4s
```

### Технологии

- Framer Motion (`useInView` для триггера)
- Stagger-анимация (задержка между карточками)
- Lottie/Rive для визуалов (или SVG-анимации)

### Плюсы

- ✅ Эффект "эволюции" (карточки появляются по порядку)
- ✅ Не требует кликов (пассивное восприятие)
- ✅ Легко на мобилке

### Минусы

- ⚠️ Менее интерактивно (нет кликов)
- ⚠️ Требует визуалов для каждой карточки

---

## Вариант 3: Tabs с визуалами (как в Tesla)

**Концепция:** 3 таба сверху. Контент меняется при клике. Справа — большой визуал.

### Визуал

```
[Культура] [Цифровой суверенитет] [Рациональная защита]
   ↓ активный
┌─────────────────────────────────────────────────────────┐
│ Для тех, кто ценит инженерную точность.                 │
│                                                          │
│ • Вы понимаете ценность каждого заводского шва          │
│ • Стандарт инсталляции без разбора дверей               │
│ • Сохранение целостности узлов                          │
│                                                          │
│         [Визуал: 3D-модель двери]                       │
└─────────────────────────────────────────────────────────┘
```

### Технологии

- React State для переключения табов
- Framer Motion для плавной смены контента
- Spline/Three.js для 3D (или статичные изображения)

### Плюсы

- ✅ Компактно (все 3 карточки в одном блоке)
- ✅ Интерактивность (клик по табам)
- ✅ Премиальный вид (как Tesla)

### Минусы

- ⚠️ На мобилке табы могут быть мелкими
- ⚠️ Требует визуалов для каждого таба

## Вариант 4: Timeline (как в Notion)

**Концепция:** Вертикальная линия с 3 точками. При скролле — точки загораются, карточки разворачиваются.

### Визуал

```
     ●───────────────────────────────────────
     │ Культура заводской сборки
     │ Для тех, кто ценит инженерную точность.
     │ • Вы понимаете ценность...
     │
     ●───────────────────────────────────────
     │ Цифровой суверенитет
     │ Ваш автомобиль всегда остается онлайн.
     │ • Вы цените точность ADAS...
     │
     ●───────────────────────────────────────
     │ Рациональная защита
     │ Физика процесса вместо временных эффектов.
     │ • Вы инвестируете в результат...
```

### Технологии

- Framer Motion (`useScroll` для прогресса линии)
- `useInView` для триггера карточек
- SVG для линии и точек

### Плюсы

- ✅ Эффект "эволюции" (траектория вниз)
- ✅ Визуально показывает прогресс
- ✅ Легко на мобилке

### Минусы

- ⚠️ Может выглядеть как "инструкция" (не премиально)
- ⚠️ Менее wow-эффекта, чем Вариант 1

## Вариант 5: Stacked Cards с Parallax (как в Stripe)

**Концепция:** 3 карточки друг за другом. При скролле — верхняя уезжает вверх, следующая выезжает.

### Визуал

```
[Скроллишь вниз]
     ↓
[Культура заводской сборки] ← уезжает вверх
     ↓
[Цифровой суверенитет] ← выезжает из-под неё
     ↓
[Рациональная защита] ← выезжает последней
```

### Технологии

- Framer Motion (`useScroll`, `useTransform`)
- `position: sticky` для эффекта "стека"
- Glassmorphism для карточек

### Плюсы

- ✅ Wow-эффект (карточки "листаются")
- ✅ Эффект "эволюции" (последовательность)
- ✅ Премиально (как в Stripe, Apple)

### Минусы

- ⚠️ Сложнее в реализации
- ⚠️ На мобилке может быть менее эффектно

---

## Моя рекомендация: Вариант 1 (Expandable Cards)

**Почему:**

- ✅ Интерактивность (клик → детали)
- ✅ Премиальный эффект (как в Apple)
- ✅ Подходит под Identity-Based (каждая карточка — отдельная "идентичность")
- ✅ Можно добавить визуалы (3D-модели, иконки, иллюстрации)

**Альтернатива:** Вариант 5 (Stacked Cards) — если хочешь wow-эффект без кликов.

## Визуалы для карточек (что использовать вместо клавиатуры/молнии)

### Карточка №1: Культура заводской сборки

**Визуал:** 3D-модель двери с подсветкой узлов (или схема дверной карты)  
**Альтернатива:** Иконка "Shield" (щит) с анимацией пульсации

### Карточка №2: Цифровой суверенитет

**Визуал:** Анимация сигналов (GPS, 5G) вокруг автомобиля  
**Альтернатива:** Иконка "Radio" (волны) с анимацией распространения

### Карточка №3: Рациональная защита

**Визуал:** График стабильности (линия без выгорания vs линия с выгоранием)  
**Альтернатива:** Иконка "Sun" (солнце) с блокировкой IR-лучей

# copyLinare Project

## Overview
A Next.js project based on the Linear landing page template, being customized for a detailing/tinting studio (ze.studio).

## Tech Stack
- Next.js
- Tailwind CSS
- TypeScript

## Development
- Run `npm install` to install dependencies.
- Run `npm run dev` to start the development server on http://localhost:3000.

## Constraints
- Preserve the original Linear design (fonts, spacing, animations, grid) at all costs.
- Replace content only based on specific instructions or documents.
- Use Russian for all content replacements as per user preference.

# Концепция блоков в стиле Linear Plan

## Референс: <https://linear.app/plan>

### Что там есть

**Структура:**

1. **Заголовок секции** (крупный, центр)
2. **Подзаголовок** (описание)
3. **Карточки 2x2** (или 1x2) с:
   - Анимированный визуал (графики, схемы, интерфейсы)
   - Заголовок карточки
   - Описание (1-2 предложения)

**Визуалы:**

- **Графики** (линии, области, столбцы) — Canvas/SVG-анимации
- **Схемы** (flowchart, roadmap) — SVG с анимацией появления
- **Интерфейсы** (мокапы Linear) — скриншоты с hover-эффектами
- **Иконки** (красный крест "Вне пути", зелёная галочка)

**Анимации:**

- Появление при скролле (fade-in + slide-up)
- Hover — лёгкий scale (1.02)
- Графики — рисуются постепенно (stroke-dasharray)

## Адаптация для ZE Studio

### Секция: "Инженерный подход к тонировке"

**Заголовок:**
> Инженерный подход к тонировке  
> Для владельцев, которые понимают ценность технологий.

**Карточки (2x2):**

#### Карточка 1 (большая, левая): Культура заводской сборки

**Визуал:** Схема дверной карты (SVG) с подсветкой узлов  
**Заголовок:** Культура заводской сборки  
**Описание:** Для тех, кто ценит инженерную точность. Стандарт инсталляции без разбора дверей — особенно для Zeekr, Li Auto, Monjaro.

#### Карточка 2 (маленькая, правая верх): Цифровой суверенитет

**Визуал:** Анимация сигналов (GPS, 5G) вокруг авто (Canvas)  
**Заголовок:** Цифровой суверенитет  
**Описание:** Ваш автомобиль всегда остается онлайн. Rayno S9 nano-ceramic — полная radio-transparency.

#### Карточка 3 (маленькая, правая низ): Рациональная защита

**Визуал:** График стабильности (линия без выгорания vs с выгоранием)  
**Заголовок:** Рациональная защита  
**Описание:** Физика процесса вместо временных эффектов. 92% IR-блокировка = -15°C в салоне.

#### Карточка 4 (большая, правая): Гарантия 10 лет

**Визуал:** Печать/сертификат с анимацией свечения  
**Заголовок:** Гарантия 10 лет  
**Описание:** Безусловная гарантия на работу и плёнку Rayno. Не выгорает, не мутнеет.

## Варианты визуалов (что проще реализовать)

### Вариант A: SVG-схемы (простой)

- **Дверная карта:** Статичный SVG с подсветкой узлов (CSS-анимация)
- **Сигналы:** SVG-волны с анимацией (keyframes)
- **График:** SVG-линия с stroke-dasharray
- **Печать:** SVG-иконка с glow-эффектом

**Плюсы:** Легко, быстро, не требует библиотек  
**Минусы:** Менее wow-эффекта

---

### Вариант B: Canvas-анимации (средний)

- **Дверная карта:** Canvas с рисованием узлов
- **Сигналы:** Canvas с частицами (как в Linear)
- **График:** Canvas с плавным рисованием линии
- **Печать:** Canvas с glow-эффектом

**Плюсы:** Больше wow-эффекта, плавные анимации  
**Минусы:** Требует кода на Canvas API

---

### Вариант C: Lottie-анимации (сложный)

- Все визуалы — готовые Lottie-файлы (JSON)
- Можно найти на LottieFiles или заказать

**Плюсы:** Максимальный wow-эффект, профессионально  
**Минусы:** Требует готовых файлов или дизайнера

---

### Вариант D: Статичные изображения + hover (самый простой)

- **Дверная карта:** PNG/WebP с подсветкой на hover
- **Сигналы:** PNG/WebP с анимацией через CSS
- **График:** PNG/WebP
- **Печать:** PNG/WebP с glow на hover

**Плюсы:** Очень быстро, не требует кода анимаций  
**Минусы:** Менее интерактивно

---

## Моя рекомендация

**Вариант A (SVG-схемы)** для старта:

1. Быстро реализуется
2. Легко адаптируется
3. Достаточно премиально для MVP

**Потом можно апгрейдить до Варианта B (Canvas)** для wow-эффекта.

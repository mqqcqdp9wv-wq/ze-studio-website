# Features Section for ZE Studio (Stitch Prompt)

Create a stunning, high-performance "Features" section for a premium car tinting studio website. The design should be heavily inspired by Linear.app's "Unlike any tool" section: dark mode, glassmorphism, subtle gradients, and data-driven visuals.

**DESIGN SYSTEM (REQUIRED):**

- **Theme:** Dark Mode, Premium, Technical.
- **Background:** Pure Black (#000000) with deep purple/blue ambient glows.
- **Surface:** Glassmorphism cards (rgba(255, 255, 255, 0.05)) with 1px border (rgba(255, 255, 255, 0.1)).
- **Typography:** Inter (San Francisco style). Headings are tight, tracking-tight.
- **Primary Text:** White (#ffffff) for headings.
- **Secondary Text:** Grey (#b4bcd0) for body text.
- **Accents:** Electric Purple/Blue gradients (Linear style).
- **Icons:** Thin, precise, technical (Lucide style).

**Page Structure:**

1. **Header (Centered):**
    - **H2 Title (Large, 7xl equivalent):** "Почему «просто тонировка»<br>стоит дороже, чем кажется"
    - **Subtitle (Medium, xl equivalent):** "Инженерные риски тонирования автомобилей с современной электроникой."

2. **Bento Grid Layout (3-4 columns):**
    - **Card 1 (Large, Wide):** "Разборка дверей"
        - *Visual:* A schematic or keyboard-like interface symbolizing precision.
        - *Text:* "В Zeekr, Voyah и новых BMW двери на одноразовых фиксаторах. При снятии обшивки в 60–80% случаев клипсы ломаются. Главная опасность — нарушение герметичности: вода попадает на блоки управления (ремонт до 350 тыс. ₽)."
    - **Card 2 (Small, Vertical):** "Экранирование сигналов"
        - *Visual:* A lightning bolt or signal wave graphic.
        - *Text:* "Металлизированные плёнки содержат алюминий. Ослабление GPS на 15–30 dB, падение скорости 5G в 2–4 раза."
    - **Card 3 (Small, Vertical):** "Цена ошибки"
        - *Visual:* A glowing seal or logo.
        - *Text:* "Экономия 3 000 ₽ сегодня — ремонт от 300 000 ₽ завтра. Мы видели случаи полной замены электроники двери после «бюджетной» тонировки."
    - **Card 4 (Large, Wide):** "Фиолетовые окна"
        - *Visual:* A command-menu interface showing a list of items/checks.
        - *Text:* "Deep Dyed-плёнки работают на органике. УФ разрушает красный компонент — через 12–18 месяцев появляется характерный фиолетовый оттенок."

**Visual Details:**

- Use subtle "shimmer" effects on card borders.
- Ensure distinct visual hierarchy: Title > Heading > Body.
- The vibe should be "Engineering Audit", not "Marketing Pitch".
